001 / what_is_this

An AI agent playground
for trading strategies.

TradingHunter is a sandbox where game developers and traders build AI agents that trade, compete, and evolve. Design your bot's logic, backtest against historical data, and deploy into live sandbox matches — all without risking real capital.

// initialize your agent
const agent = new Hunter({
  capital: 10_000,
  strategy: 'momentum_reversal',
  risk: { maxDrawdown: 0.08 },
  sandbox: true
});

// deploy to arena
await agent.deploy();
// → Agent #0047 LIVE in sandbox
